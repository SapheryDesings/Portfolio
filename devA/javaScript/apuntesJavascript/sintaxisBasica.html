¿Como incluimos nuestro código?

En línea

En el mismo archivo HTML con la etiqueta "script", que contiene las órdenes o líneas de javascript. No es la forma recomendable, lo ideal es serparar el código en un archivo aparte del HTML.

                
                    script console.log(); /script

Archivo externo

Para relacionar un documento javascript, utilizaremos también la etiqueta script, pero en este caso incluiremos el atributo src para indicar la ruta donde se encuentra el archivo.

            
                script src="archivojavascript.js" /script

La ubicación de la etiqueta script, puede estar en diferentes sitios.

Ubicación Cuando descarga el archivo Estado de la página
En la etiqueta head Antes de empezar a dibujar la página Página aún no dibujada
En la etiqueta body Durante el dibujado de la página Dibujada hasta donde está la etiqueta script
Antes del /body Después de dibujar la página Dibujada al 100%

Elegiremos el sition que más nos convenga para incluir la etiqueta script

Operadores

Aritméticos

Para realizar operaciones matemáticas.

Nombre Operador Descripción
Suma a + b Suma el valor de a al valor de b
Resta a - b Resta el valor de b al valor de a
Multiplicación a * b Multiplica el valor de a por el valor de b
División a / b Divide el valor de a entre el valor de b
Módulo a % b Devuelve el resto de la división de a entre b
Exponenciación a ** b Eleva a a la potencia de b, es decir, ab. Equivalente a Math.pow(a, b)

Asignación

Para guardar información en variables

Nombre Operador Descripción
Asignación c = a + b Asigna el valor de la parte derecha a c
Suma y asignación a += b Es equivalente a a = a + b
Resta y asignación a -= b Equivalente a a = a - b
Multiplicación y asignación a *= b Equivalente a = a * b
División y asignación a /= b Equivalente a a = a / b
Módulo y asignación a %= b Equivalente a = a % b
Exponenciación y asignación a **= b Equivalente a a = a ** b

Comparación

Para realizar comprobaciones

Nombre Operador Descripción
Operador de igualdad a == b Comprueba si el valor de a es igual al de b. No comprueba el tipo de dato
Operador de desigualdad a != b Comprueba si el valor de a no es igual al de b
Mayor que a > b Comprueba si el valor a es mayor que el de b
Mayor/igual que a >= b Comprueba si el valor de a es mayor o igual que el de b
Menor que a < b Comprueba si el valor de a es menor que el de b
Menor/igual que a <= b Comprueba si el valor de a es menor o igual que el de b
Operador de identidad a === b Comprueba si el valor y el tipo de dato de a es igual al de b
Operador no idéntico a != b Comprueba si el valor y el tipo de dato de a no es igual al de b

Operadores lógicos

Como trabajar con valores boolean o similares

Nombre Operador Descripción
Operador lógico AND a && b Devuelve a si es false, sino devuelve b
Operador ternario ?: a ? b : c Si a es true, devuelve b, sino devuelve c
Lógico OR a || b Devuelve a si es true, sino devuelve b
Lógico Nullish coalescing a ?? b Devuelve b si a es null o undefined, sino devuelve a
Operador de asignación lógica nula a ??= b Es equivalente a a ?? (a = b)
Operador de encadenamiento opcional data?.name Perminte intentar acceder a una propiedad, aunque su padre no exista
Operador unario lógico NOT !a Invierte el valor. Si es true devuelve false y viceversa
