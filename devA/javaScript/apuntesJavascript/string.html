Tipo de dato String

Una de las tareas que más solemos hacer es menejar cadenas de texto y realizar tareas con ellas. Cuando hablamos de una variable que posee información de texto, decimos que su tipo de dato es string.

Constructor Descripción
new String(text) Crea un objeto de texto a partir del texto text pasado por parámetro. Evitar
"texto" Simplemente, escribimos el texto entre comillas simples o dobles. Notación preferida.

Los string son uno de los tipos de datos primitivos, es más sencillo de utilizar los literales que la notación que utiliza la palabra clave new. Se pueden utilizar tres tipos de comillas:

  • Simples: ''
  • Dobles: ""
    • Backticks: `` (Interpolación de variables)

Se recomienda elegir por uno de los estilos y no mezclarlos.

          
            // Notación literal (preferida)
            const text = "¡Hola a todos!";
            const message = "Otro mensaje de texto";

            // Notación mediante objeto
            const text = new String("¡Hola a todos!");
            const message = new String("Otro mensaje de texto");

Propiedades de un string

Al crear un string esa variable pasa a tener a su disposición todas las propiedades y métodos disponibles.

Propiedad Descripción
.length

Se encargade devolver el tamaño total de la cadena.

          
            "Hola".length;  //4
            "Adiós".length; //5
            "".length;      //0
            "¡Yeah!".length //6

Acceso a un carácter

Un string esta formado por carácteres, como si de un array se tratase. Si queremos acceder a cada uno de ellos, podemos utilizar el operador [] indicando la posición a la que queremos acceder.

          
            const text = "Hola";
            text[0];  //"H"
            text[1];  //"o"
            text[2];  //"l"
            text[4];  //undefined

Si intentamos acceder a una posición que no existe, nos devolverá undefined.

Métodos de un string

Las carácteristicas más interesantes se encuentran en los métodos. En las variables de texto los métodos se ejecutan sobre el propio texto del string, devolviendo información modificada a partir de este.

El método .repeat() está disponible en cualquier string, y repite el texto el número de veces que le indicamos por parámetro:

          
            const text = 'Javascript';

            text.repeat(3); //"JavascriptJavascriptJavascript"

Interpolación de variables

A partir de ES2015, se introducen las backticks (comillas hacia atrás), que nos permitirán interpolar el valor de las variables sin tener que cerrar, concatenar y abrir la cadena de texto contituamente:

          
            const primeraCadena = "frase";
            const segundaCadena = "concatenada";
            `Una ${primeraCadena} mejor ${segundaCadena}`; // 'Una frase mejor concatenada'

Es una funcionalidad muy simple que mejora sustancialmente la legibilidad de código. Aporta una serie de ventajas:

  • Permite múltiples líneas, algo que no se puede hacer con las demás comillas.
  • Permite interpolar expresiones Javascript, no solo variables.
  • Permite interpolar el valor de variables.

Esto es ideal para crear plantillas reutilizables:

          
            const palabra = `strong palabra /strong`;
            const plantilla = `
              div class = "container"
              &{palabra}
              div
            `;

La plantilla contiene HTML de palabra, que cobra sentido para crear funciones reutilizables:

          
            function crearBoton(className, text){
              return`button class="${className}"${text} /button`;
            }

            crearBoton("primary", "Aceptar");
            crearBoton("warning", "Eliminar");
