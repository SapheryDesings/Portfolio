Conversiones numéricas

Que no tengamos que declarar su tipo de dato, no significa que nos despreocupemos de ello. Hay momentos en que nos interese convertir un string a número por muchos motivo.

Convertir texto a número. Números enteros

Para convertir un string en number lo ideal es utilizar las funciones de parseo numérico, parseInt() y parseFloat().

Método Descripción
Number.parseInt(text) Convierte un string text en un número entero
Number.parseInt(text, radix) Idem, pero el string un número en base número radix

Ejemplos con el método .parseInt() cuando sólo le pasamos un parámetro (texto) que queremos convertir a número.

            
                Number.parseInt("42");          //42
                Number.parseInt("42€");         //42
                Number.parseInt("Núm. 42");     //NaN
                Number.parseInt("A");           //NaN
                Number.parseInt("");            //NaN

.parseInt() funciona bien para variables de texto que son números (o que empiezan por números), muy útil para eliminar unidades de variables de texto que se extraen de una página. Si la variable comienza por un valor que no es númerico, descartará todo el texto y devolverá un NaN. Si el string contiene números decimales, sólo se quedará con la parte entera.

            
                Number.parseInt("42.5");        //42
                Number.parseInt("88.99€");      //88
                Number.parseInt("Núm. 35.5");   //NaN

Números decimales

.parseFloat() funciona igual que .parseInt() pero con números decimales.

Método Descripción
Number.parseFloat(text) Convierte un string text en un número decimal
Number.parseFloat(text, radix) Idem, pero el string tiene un número en base número radix. 
            
                Number.parseFloat("42.5");      //42.5
                Number.parseFloat("42");        //42
                Number.parseFloat("88.99€");    //88.99
                Number.parseFloat("42€");       //42
                Number.parseFloat("Núm. 33.5"); //NaN

Convertir desde otra base

Los método anteriores tiene un segundo parámetro number radix, que sirve para indicar la base numérica desde la que procede el número del string. Cuando se omite este parámetro, se asume que la base es 10 (base decimal).

            
                // Base binaria: 0, 1 (Base 2)
                // Base octal: 0, 1, .. 7 (Base 8)
                // Base decimal: 0, 1, ..9 (Base 10)
                // Base hexadecimal: 0, 1, .. 9, A, B, C, D, E, F (Base 16)

Convertir a decimal

Podemos convertir de cualqueier base numérica a la decimal, utilizando el parámetro radix.

            
                Number.parseInt("11101", 2);    // 11101 en binario, es 29 en decimal
                Number.parseInt("31", 8);       // 31 en octal, es 25 en decimal
                Number.parseInt("FF", 16);      // FF en hexadecimal, es 255 en decimal

Otra forma de representar estos números en una base específica es utilizando la nomenclatura de la base numérica mediante un prefijo 0b, 0o o 0x. Al utilizarlos, se convertirán automáticamente a base decimal.

            
                0b11101     // 29
                0o31        // 25
                0xFF        // 255

Convertir desde decimal

También podemos realizar la operación inversa, de base decimal a otra base numérica diferente, utilizando el método .toString() de los string.

Método Descripción
Number.toString() Convierte un number en string
Number.toString(radix) Convierte un number a la base number radix y lo devuelve como texto 
            
                (16).toString();        // "16"
                (42.5).toString();      // "42.5"
                (26).toString(2);       // "11010"
                (80).toString(8);       // "120"
                (245123).toString(16);  // "3bd83"
