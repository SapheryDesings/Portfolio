Objeto Math

En javascript es posible realizar gran cantidad de operaciones matemáticas de forma nativa, sin necesidad de librerías externas. Para ello utilizamos el objeto Math, que es un objeto interno que tiene incorporadas ciertas constantes y métodos (funciones).

Constantes de Math

Constante Descripción Valor
Math.E Número de Euler 2.718281828459045
Math.LN2 Equivalente a Math.log(2) 0.6931471805599453
Math.LN10 Equivalente a Math.log(10) 2.302585092994046
Math.L0G2E Equivalente a Math.log2(Math.E) 1.4426950408889634
Math.LOG10E Equivalente a Math.log10(Math.E) 0.4342944819032518

Math también nos proporciona gran cantidad de métodos o funciones.

Métodos matemáticos

Método Descripción Ejemplo
Math.abs(x) Devuelve el valor absoluto de |x| Math.abs(-5); // 5
Math.sign(x) Devuelve el signo del número: 1 positivo, -1 negativo Math.sign(-5); // -1
Math.exp(x) Exponenciación. Devuelve el número e elevado a x // ex Math.exp(1); // e
Math.expm1(x) Equivalente a Math.exp(x) - 1 // ex-1 Math.expm1(1); // 1.718281828459045
Math.max(a, b, c...) Devuelve el número más grande de los indicados por parámetro Math.max(1, 40, 5, 15); // 40
Math.min(a, b, c...) Devuelve el número más pequeño de los indicados por parámetro Math.min(5, 10, -2, 0); // -2
Math.pow(base, exp) Potenciación. Devuelve el número base elevado a exp // baseexp Math.pow(2, 10); // 1024
Math.sqrt(x) Devuleve la raíz cuadrada de x Math.sqrt(2); // 1.4142135623730951
Math.cbrt(x) Devuelve la raíz cúbica de x Math.cbrt(2); // 1.2599210498948732
Math.imul(a, b) Equivalente a a * b, pero a nivel de bits Math.imul(0xffffffff, 7); // -7
Math.clz32(x) Devuelve el número de ceros a la izquierda de x en binario (32 bits) const x = 1;
"0".repeat(Math.clz32(x)) + x.toString(2);

El método Math.random() lo explicaremos más detalladamente.

Método Math.random()

Método Descripción
Math.random() Devuelve un número al azar entre 0 y 1 con 16 decimales.

Normalmente, necesitamos buscamos obtener un número entero al azar entre a y b.

            
                // Obtenemos un número al azar entre [0, 1] con 16 decimales
                let x = Math.random();

                // Multiplicamos dicho número por el valor máximo que buscamos (5)
                x = x * 5;

                // Redondeamos inferiormente, quedándonos sólo con la parte entera
                x = Math.floor(x);

El ejemplo dará un vlaor al azar entre 0 y 5, este no incluido

            
                // Número al azar entre 0 y 5 (no incluido)
                const x = Math.floor(Math.random() * 5);

                // Equivalente al anterior
                const x = ~~(Math.random() * 5);

En este segundo ejemplo, utilizamos el operador a nivel de bits ~~ (doble negación) como reemplazo rápido de Math.floor(), función que realiza un redondeo inferior.

Métodos de logaritmos

Método Descripción Ejemplo
Math.log(x) Devuelve el logaritmo natural en base e de x loge x o ln x
Math.log10(x) logaritmo decimal (en base 10) de x log10 x o log x
Math.log2(x) logaritmo binario (en base 2) de x log2 x
Math.log1p(x) logaritmo natural de (1+x) loge(1+x) o ln (1+x)

Ejemplos:

            
                Math.log(2);    // 0.6931471805599453
                Math.log10(2);  // 0.3010299956639812
                Math.log2(2);   // 1
                Math.log1p(2);  // 1.0986122886681096

Método de redondeo

Es muy común necesitar métodos para redondear números y reducir el número de decimales o aproximar a una cifra concreta.

Método Descripción
Math.round(x) Devuelve x con redondeo (el entero más cercano)
Math.ceil(x) Devuelve x con redondeo superior (el entero más alto)
Math.floor(x) Devuelve x con redondeo inferior (el entero más bajo)
Math.fround(x) Devuelve x con redondeo (flotante con precisión simple)
Math.trunc(x) Trunca el número x (devuelve sólo la parte entera)

Ejemplos

            
                // Redondeo natural, el más cercano
                Math.round(3.75);           // 4
                Math.round(3.25);           // 3

                // Redondeo superior (el más alto)
                Math.ceil(3.75);            // 4
                Math.ceil(3.25);            // 4

                // Redondeo inferior (el más bajo)
                Math.floor(3.75);           // 3
                Math.floor(3.25);           // 3

                // Redondeo con precisión
                Math.round(3.123456789);    // 3
                Math.fround(3.123456789);   // 3.1234567165374756

                // Truncado (sólo parte entera)
                Math.trunc(3.75);           // 3
                Math.round(-3.75);          // -4
                Math.trunc(-3.75);          // -3

Método Descripción

Math.asin(x) Arcoseno de x
Math.sinh(x) Seno hiperbólico de x
Math.asinh(x) Arcoseno hiperbólico de x
Math.cos(x) Coseno de x
Math.acos(x) Arcocoseno de x
Math.cosh(x) Coseno hiperbólico de x
Math.acosh(x) Arcocoseno hiperbólico de x
Math.tan(x) Tangente de x
Math.atan(x) Arcotangente de x
Math.tanh(x) Tangente hiperbólica de x
Math.atanh(x) Arcotangente hiperbólica de x
Math.atan2(x, y) Arcotangente del cociente de x/y
Math.hypot(a, b..) Devuelve la raíz cuadrada de a2 + b2 + ...

Al contrario que muchos otros objetos, en estas ocasiones hemos indicado explícitamente el objeto (Math.round(numero)), en lugar de hacerlo sobre la variable (numero.round()). Esto ocurre porque Math es un objeto con métodos y constantes estáticas, algo que veremos en profundidad en futuros temas.
