NaN - Not a number

NaN es un valor especial, que significa Not a number (No es un número), aunque no es su significado literal. Se usa para representar valores numéricos que son indeterminados o imposibles de representar como número.

  • Indeterminación matemática: Ejemplo 0 / 0
  • Valores imposibles: Ejemplo 4 - 'a'
  • Operaciones con NaN: Ejemplo NaN + 4

Propiedades y métodos relacionados con NaN

Propiedad o Método Descripción
Number.NaN Es equivalente a NaN. Valor que no puede representarse como número.
Number.isNaN(number) Comprueba si number

Observa que existe Number.NaN como propiedad de number, para tenerlo modularizado y encapsulado como propiedad estática en el objeto Number.

El valor de NaN se utilizar para intentar representar un valor que no se puede representar, así que no podemos compararlo literalmente con NaN, sino que tenemos que utilizar la función Number.isNaN().

            
                Number.isNaN(NaN);  //true
                Number.isNaN(5);    //false
                Number.isNaN("A");  //false

Recuerda que NaN sí que es un valor numérico no representable de forma númerica.

            
                const number = NaN;

                typeof number;
                number.constructor.name;

Conversiones NaN

Si intentamos convertir un valor NaN a otro tipo de dato primitivo utilzando un typecast(forzar conversión explícita)

            
                Boolean(NaN);   //false
                String(NaN);    //"NaN"
                Number(NaN);    //NaN
