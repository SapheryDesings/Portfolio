Posiciones y substrings

Una substring es un fragmento más pequeño que forma parte de un string. También se suele hacer referencia a ellos como subcadena o subcadena de texto.

Una posición es un number que representa el lugar donde está ubicado un substring, teniendo en cuenta que se empieza a contar en 0. Así pues, la primera letra del string tendría el índice 0, la segunda 1, la tercera 2, etc...

            
                "Hola amigos"   // "amigos" es un substring
                "Hola amigos"   // la posición de "amigos" es 5
                0123456789

Obtener posición o índice

Sobre la posición o índice exsiten métodos que permiten darnos información.

Método Descripción
.charAt(pos) Devuelve el carácter de la posición pos. Similar al operador []
.indexOf(text) Devuelve la primera posición del texto text
.indexOf(text, from) Idem al anterior, partiendo desde la posición from
.lastIndexOf(text) Devuelve la última posición del texto text
.lastIndexOf(text, from) Idem al anterior, partiendo desde from hacia el inicio

Carácter en cierta posición

El método .charAt(pos) comprueba que carácter se encuentra en la posición pos del texto. Nos devuelve un string con dicho carácter. Si le pasamos una posición que no existe nos devolverá un string vacío. El valor por defecto de pos es 0, si se omite se obtendrá el primer elemento.

Es preferible utilizar el operador [] para obtener el carácter que ocupa una posición, ya que es más corto y rápido de utilizar y mucho más claro.

            
                const nombre = "Saphery";
                
                nombre.charAt();    //'S'
                nombre.charAt(0);   //'S'
                nombre.charAt(1);   //'a'
                nombre.charAt(15);  //''

                nombre[];       //ERROR
                nombre[0];      //'S'
                nombre[1];      //'a'
                nombre[15];     // undefined

Posición de cierto carácter

El método .indexOf(text, from) realiza la función opuesta a charAt(). El método .indexOf(text) buscará el subtexto text en nuestro string y nos devolverá un number con la posición de la primera aparición de dicho subtexto. Sino lo encuentra, devuelve -1. El parámetro from es opcional y es la posición en la que empezará a buscar, si se omite es 0.

            
                const cadena = "LenguajeJS, página de Javascript";

                cadena.indexOf('n');    //2
                cadena.indexOf('n, 3')  //16
                cadena.indexOf('n', 17) //-1

Posición desde el final

El método .lastindexOf(text, from) realiza la búsqueda empezando desde el final de la cadena hacia el principio.

            
                const cadena = "LenguajeJS, página de Javascript";

                cadena.lastindexOf('n');    //16
                cadena.lastindexOf('n, 3')  //2
                cadena.lastindexOf('n', 1) //-1

Obtener fragmento substrings

Métodos mediante los que podemos crear substrings formados por un fragmento del string original.

Método Descripción
.repeat(num) Devuelve el string repetido num veces
.substring(start, end) Devuelve el substring desde la posición start hasta end
